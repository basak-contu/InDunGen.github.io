InDunGen
ABOUT THE PROJECT

In today's game industry, procedural map generation algorithms frequently used for generating game levels.In project InDunGen, we wanted to try GAN-based approach for generating game levels.

InDunGen focuses on utilizing Generative Adversarial Networks (GANs) to generate maps for games, specifically focusing on the popular rouge-like game NetHack. The objective is to train GAN models to learn from existing levels and generate new, diverse levels while maintaining the game's complexity. We aimed to provide a novel approach to level design that enchances replayability and engagement for players.

METHOD DETAILS
Dataset Preparation

We collected a large dataset of existing NetHack levels as the training data for GAN models by modifying NetHack's source code. We exported level data as unsigned char array then modified them for training.

Nvidia's StyleGAN2

NVIDIA StyleGAN2 is a deep learning-based generative model developed by NVIDIA, which is designed to generate high-quality images with realistic and diverse visual content. It is an improved version of the original StyleGAN model.The main objective of StyleGAN2 is to generate highly realistic and visually appealing images that resemble a specific dataset or follow a certain style. The model achieves this by learning the underlying distribution of the training data and generating new images that capture the characteristics and variations present in the dataset.

In the project, we utilized the NVIDIA StyleGAN2 model for generating new levels in the game NetHack. The main objective was to leverage the power of generative adversarial networks (GANs) and the high-quality image generation capabilities of StyleGAN2 to create diverse and visually appealing levels. Here is one of the generated level.

GAN using Adaptive Instance Normalization

Adaptive Instance Normalization (AdaIN), aims to improve the style transfer abilities of the generator network.Normalizing the features in the generator is necessary for strong training and generating consistent outputs. Other normalization techniques, such as Batch Normalization (BN), can effectively normalize the features but remove the style information. This limitation limits the ability of GANs to produce diverse outputs. AdaIN addresses this limitation by dynamically adjusting the mean and standard deviation of the features in the generator network based on the desired style information. It achieves this by leveraging the statistics from a style reference image, which serves as a source of style information.

We also modified GAN model framework coupled with Adaptive Instance Normalization (AdaIN) for generating NetHack game levels. The GAN model consisted of two main components: a generator network and a discriminator network. The generator network was responsible for producing new NetHack levels, while the discriminator network aimed to distinguish between real and generated levels. Here is one of the generated level.

In the project, we found that AdaIN GAN did not achieve as successful results as StyleGAN2. StyleGAN2 has been specifically designed for complex images. It leverages a progressive growing architecture and incorporates several architectural enhancements that allow it to capture fine-grained details and generate visually appealing outputs.while AdaIN GAN is a promising approach for image generation tasks, its performance may be more limited in the context of NetHack level generation compared to the specialized architecture and capabilities of StyleGAN2.
